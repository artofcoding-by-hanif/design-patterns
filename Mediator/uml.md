    |      Mediator      |                |   Colleague A    |          |   Colleague B    |
    +--------------------+                +------------------+          +------------------+
    | - notify()         |<-------------->| - send()        |<-------->| - send()         |
    +--------------------+                +------------------+          +------------------+
            ^                                                             
            |                                                             
    +-------------------------+                                          
    | ConcreteMediator        |
    +-------------------------+
    | - notify(colleague)     |
    | - addColleague()        |
    +-------------------------+
